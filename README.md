## рҹҡҖ Getting Started

### Prerequisites

- **Node.js** v16 or higher
- **npm** or **yarn**
- A MikroTik router with REST API enabled on port `8728`

<br/>

### рҹ“Ұ Installation

**1. Clone the repository**

```bash
git clone https://github.com/Mahfujulislam5/MikroDesk.git
cd MikroDesk
```

**2. Install dependencies**

```bash
npm install
```

**3. Start in development mode**

```bash
npm run electron-dev
```

> This starts the Vite dev server on `http://localhost:5173` and launches the Electron app with hot reload.

**4. Build for production**

```bash
npm run electron-build
```

> Installer will be generated in the `out/` directory.

## рҹ”Ң MikroTik API Usage

```js
import MikroTikService from './services/MikroTikService';

// Connect to router
const result = await MikroTikService.connect('192.168.88.1', 'admin', 'password');

// Get all online users
const onlineUsers = await MikroTikService.getOnlineUsers();

// Create a new hotspot user
const newUser = await MikroTikService.createHotspotUser({
  name: 'testuser',
  password: 'pass123',
  profile: 'default'
});
```

### Supported Operations

- вң… Connect / Disconnect from router  
- вң… List all hotspot users  
- вң… Create / Delete users  
- вң… Get real-time online sessions  
- вң… Fetch router system info

## рҹҗӣ Troubleshooting

<details>
<summary><b>рҹ”ҙ Connection Issues</b></summary>

- Verify the router IP address is reachable on your network
- Ensure MikroTik REST API is enabled on port `8728`
- Check firewall rules on both the router and your machine
- Confirm username/password credentials are correct

</details>

<details>
<summary><b>рҹ”ҙ Database Issues</b></summary>

- To reset the database, delete: `C:\Users\{Username}\.mikrodesk\database.db`
- Rebuild the native module: `npm rebuild better-sqlite3`

</details>

<details>
<summary><b>рҹ”ҙ Electron Issues</b></summary>

- Clear Vite cache: delete the `.vite` folder
- Reinstall all dependencies:
  ```bash
  rm -rf node_modules
  npm install
  ```

</details>
